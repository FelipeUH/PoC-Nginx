¿Qué es esta prueba de concepto?

Esta es una prueba de concepto para demostrar una arquitectura básica de software utilizando NGINX como servidor de proxy inverso. El objetivo es mostrar cómo podemos utilizar NGINX para dirigir las solicitudes a diferentes APIs (como usuarios y productos) y servir archivos estáticos (HTML, CSS, JavaScript).

Componentes de la arquitectura

  • NGINX como servidor web.
  • NGINX como proxy inverso.
  • NGINX como balanceador de cargas.
